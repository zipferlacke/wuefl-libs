01 — Installation

Einbinden

Einfach als ES-Modul importieren. Das CSS wird automatisch nachgeladen — selectpicker.css muss im selben Verzeichnis liegen.

import { SelectPicker } from './selectpicker.js';

Schnellstart

<!-- HTML -->
<select id="meinSelect">
  <option value="a">Option A</option>
  <option value="b">Option B</option>
</select>

// JavaScript
const selectpicker = new SelectPicker({});
selectpicker.create(document.getElementById('meinSelect'), {
  title: 'Option wählen'
});

// Auf Änderungen reagieren
document.getElementById('meinSelect')
  .addEventListener('change', (e) => {
    const vals = Array.from(e.target.selectedOptions).map(o => o.value);
    console.log(vals);
  });
02 — API

Konfiguration

new SelectPicker(config)

ParameterTypBeschreibung
config.langstringStartsprache: 'de' oder 'en'. Standard: Auto-Detect via Browser.
config.translationsObjectEigene Übersetzungen für Buttons und Platzhalter.
config.optionsObjectGlobale Standard-Optionen für alle Picker dieser Instanz.

selectpicker.create(selectElement, options)

OptionTypStandardBeschreibung
titlestring | {de,en}"Option wählen"Titel im Popover-Header.
groupstring"default"Name der Style-Gruppe (via createGroup).
searchbooleantrueSuchfeld anzeigen.
saveButtonbooleanfalseSpeichern-Button. Empfohlen bei Multiselect.

Mehrsprachige Titel

selectpicker.create(selectElement, {
  title: { de: 'Sprache wählen', en: 'Choose language' }
});

Die SelectInstance

Methode / EigenschaftBeschreibung
instance.open()Öffnet das Popover programmatisch.
instance.close()Schließt und speichert.
instance.saveAndClose()Speichert die Auswahl und schließt.
instance.reset()Setzt auf Standardwerte zurück.
instance.isOpentrue wenn das Popover geöffnet ist.
instance.isMultitrue bei multiple Select.
instance.originalSelectReferenz auf das originale <select>.

Icons per data-icon

Jede <option> kann ein Icon erhalten — entweder als Material Symbol Name oder als SVG-String:

<option value="home" data-icon="home">Startseite</option>
<option value="x" data-icon='<svg viewBox="0 0 24 24">...</svg>'>Eigenes Icon</option>

Standardwerte & Reset

<select id="multi" multiple data-default='["a","b"]'>
  <option value="a" selected>Alpha</option>
  <option value="b" selected>Beta</option>
</select>
info

Der Reset-Button erscheint nur wenn data-default gesetzt ist.

Eigene Übersetzungen

const selectpicker = new SelectPicker({
  lang: 'de',
  translations: {
    de: { save: "Speichern", search: "Suche...", reset: "Zurücksetzen", empty: "Nichts gewählt...", search_nodata: "Keine Daten..." },
    en: { save: "Save", search: "Search...", reset: "Reset", empty: "Nothing selected...", search_nodata: "Nothing found..." }
  }
});
03 — CSS

Anpassen

Alle visuellen Aspekte werden über CSS Custom Properties gesteuert. Light/Dark Mode wird automatisch via light-dark() unterstützt.

VariableStandardBeschreibung
--bg-pickerlight-dark(hsl(0,0%,90%), hsl(0,0%,10%))Hintergrund des Popovers
--bg-picker-triggerlight-dark(...)Hintergrund des Triggers
--clr-pickerlight-dark(...)Textfarbe
--fs-picker-header1remSchriftgröße Titel
--fs-input-picker0.9remSchriftgröße Optionen & Input
--br-picker0.5remBorder-Radius überall
--height-input-picker2remHöhe Buttons & Suchfeld
--bg-input-pickerhsl(219,100%,50%)Farbe Buttons & Suchfeld-Border
--bg-hover-input-pickerhsl(219,100%,60%)Hover-Farbe Buttons
--bg-option-pickerhsl(0,0%,80%)Hintergrund einer Option
--bg-hover-option-pickerhsl(0,0%,85%)Hover-Hintergrund Option
--bg-selected-option-pickerhsl(219,100%,50%)Selektierte Option
--bg-selected-hover-option-pickerhsl(219,100%,55%)Selektierte Option Hover

Globale Resets & @layer

Das Picker-CSS liegt in @layer components. Wenn deine Seite einen ungelayerten * { padding: 0 } Reset hat, pack ihn ebenfalls in einen Layer:

@layer resets, components;
@layer resets {
  * { padding: 0; margin: 0; box-sizing: border-box; }
}

CSS Anchor Positioning

In Browsern mit Unterstützung (Chrome 125+, Firefox 147+) wird die Popover-Position rein per CSS gesteuert. Ältere Browser erhalten automatisch einen JS-Fallback via getBoundingClientRect.
